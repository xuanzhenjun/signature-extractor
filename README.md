# 签名背景去除工具

免费在线去除签名背景，生成透明PNG图片。

## 功能特点

- ✅ 完全免费，无需注册
- ✅ 浏览器本地处理，保护隐私
- ✅ 支持拖拽上传
- ✅ 实时预览调整
- ✅ 一键下载透明PNG

## 本地运行

直接打开 `index.html` 即可使用。

## Cloudflare Pages 部署

1. 将代码推送到 GitHub
2. 登录 Cloudflare Dashboard
3. Pages → Create a project → Connect to Git
4. 选择仓库，配置如下：
   - **Build command**: 留空
   - **Build output directory**: `/`
5. 点击 Deploy

## 技术栈

- 纯 HTML/CSS/JavaScript
- Tailwind CSS (CDN)
- Canvas API
